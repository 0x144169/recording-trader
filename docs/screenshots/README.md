# 产品截图目录

将主要界面的截图放在此目录，并在仓库根目录的 **README.md** 中引用，这样访客打开仓库即可看到产品样子。

**建议截图（与 README 界面预览对应）：**

| 文件名 | 说明 |
|--------|------|
| `dashboard.png` | 仪表盘 / 交易概览 |
| `journal.png` | 订单总览 / 日历 |
| `review-mode.png` | 复盘模式（左侧列表 + 右侧详情） |

| `settings.png` | 设置页 |

| `daily-report.png` | 日报告：点击某日后该日编辑面板（对比图、笔记、评分） |
| `share-card-entry.png` | 晒单入口：日报告订单卡片右下角「晒」按钮所在页面 |

| `knowledge-linked-trade.png` | 知识库关联订单跳转（跨模板定位） |
