# 产品截图目录

将主要界面的截图放在此目录，并在仓库根目录的 **README.md** 中引用，这样访客打开仓库即可看到产品样子。

**建议截图（与 README 界面预览对应）：**

| 文件名 | 说明 |
|--------|------|
| `dashboard.png` | 仪表盘 / 交易概览 |
| `journal.png` | 订单总览 / 日历 |
| `review-mode.png` | 复盘模式（左侧列表 + 右侧详情） |
| `learning.png` | 学习园 |
| `settings.png` | 设置页 |
| `new-order.png` | 新订单弹窗 |
| `export-learning.png` | 导出学习 Zip 弹窗（筛选、脱敏等选项） |
| `image-preview.png` | 做单/复盘图点开大图预览（含左右切换） |
| `daily-report.png` | 日报告：点击某日后该日编辑面板（对比图、笔记、评分） |
| `share-card-entry.png` | 晒单入口：日报告订单卡片右下角「晒」按钮所在页面 |
| `share-card-modal.png` | 晒单选项弹窗：选图、隐私选项、预览、保存/复制 |
| `share-card-result.png` | 晒单分享图：生成后的宽幅图（K 线图 + 右侧文字竖条） |

截图后放入本目录，README 中已按上述文件名引用，会以 4 行 3 列显示。
