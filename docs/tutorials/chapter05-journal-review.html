第 05 章：订单总览与复盘模式

到这一章，记录不再只是留档，而是正式进入复盘。你会看到如何在 Journal 里先筛出值得看的样本，再进入 Review Mode，把截图、评分、情绪标记和复盘文字整理成真正可复用的经验。

看完你应该能做到

  • 在 Journal 用筛选和卡片流快速缩小复盘样本范围。
  • 进入 Review Mode 查看订单细节、图片证据和复盘文字。
  • 保存已复盘状态，让订单从历史记录变成经验样本。

章节提要

内容结构

  • 先筛样本：用品种、结果、时间和标签把范围缩小。
  • 进入复盘：从 Journal 进入 Review Mode，切到结构化工作台。
  • 看图说话：点开截图预览，优先用证据而不是只靠回忆。
  • 保存已复盘：留下状态、评分和结论，形成可回看的经验样本。

适合谁看

  • 已经开始持续录单，准备建立稳定复盘节奏的人。
  • 不想只存历史，而想把订单沉淀成经验的人。
  • 需要用截图、执行细节和结论回看真实交易过程的人。
