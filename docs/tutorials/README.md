# Tutorials on GitHub Pages

This folder is the public tutorial landing area for `recording-trader`.

## Structure

- `index.html`: tutorial homepage
- `chapter01-install-download.html`: chapter 01 landing page
- `videos/`: final MP4 files served by GitHub Pages
- `posters/`: chapter cover images shown in README and pages
- `assets/tutorials.css`: shared page styles

## Publish workflow

1. Render the chapter video in the private tutorial repo.
2. Copy the final MP4 into `docs/tutorials/videos/`.
3. Copy the chapter poster into `docs/tutorials/posters/`.
4. Add or update the chapter page and homepage card.
5. Ensure GitHub Pages is enabled with:
   - Branch: `main`
   - Folder: `/docs`

## Expected public URLs

- Tutorials homepage: `/tutorials/`
- Chapter page: `/tutorials/chapter01-install-download.html`
- Direct MP4: `/tutorials/videos/chapter01-install-download.mp4`
