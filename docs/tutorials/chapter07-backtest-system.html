第 07 章：系统回测

这一章不讲零散回看，而是讲一套真正能长期复用的回测系统。你会看到如何把策略样本、结构化记录、 单笔智能分析和统计比较组织到同一条工作流里，让回测从“看胜率”升级为“打磨方法”。

看完你应该能做到

  • 先从总览判断回测系统有没有形成样本面。
  • 把单笔样本录成统一结构，便于后续比较和复盘。
  • 回到统计页判断系统是否稳定，而不是只盯单笔输赢。

章节提要

内容结构

  • 看系统总览：策略文件夹、记录列表和结果一起出现，先建立系统感。
  • 录成结构：关键位、入场、止盈止损、CISD、清扫和时间字段都要写清。
  • 用智能分析：让分析建立在你已经写好的结构字段上。
  • 回到统计页：通过排序、筛选和样本比较判断方法是不是稳定。

适合谁看

  • 不想只看几笔输赢，而想系统化打磨策略的人。
  • 想把回测样本做成可以比较和统计的方法库的人。
  • 需要把智能分析和结构化字段结合使用的人。
